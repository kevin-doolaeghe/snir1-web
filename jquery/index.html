Technologies AJAX & JQuery

AJAX

  • AJAX est une technologie Web pour exécuter des morceaux de code PHP sans recharger complètement la page. On peut aussi manipuler le DOM et utiliser Javascript.
  • AJAX est l’acronyme d’Asynchonous Javascript and XML
  • Il rend les sites Web dynamiques & intéractifs
  • On peut utiliser AJAX avec l’objet XMLHttpRequest en Javascript ou à l’aide de la bibliothèque JQuery


JQuery

  • JQuery est une bibliothèque Javascript lancée en 2006 par John Resig
  • Il sert à :
    • simplifier la manipulation du DOM
    • simplifier la gestion des évènements
    • ajouter des effets visuels & des animations
    • manipuler les feuilles de style
    • manipuler des plugins
    • manipuler AJAX
